Best Practices for Using CSS Selectors: ID vs Class

One can define your own HTML selectors in the form of Class and ID selectors. The benefit of this is that you can have the same HTML element, but present it different depending on its class. When deciding on which selector to use you need should consider these three things:

1. Repeated use within the document

ID is used for uniquely identifying elements and can ony be applied once within a document. Class is used for identifying an element as being part of a class of elements and can be used as many times as needed within a document.

2. Combining class selectors

3. IDs have a higher specificity than classes

Meaning, IDs carry a heavier weight of specificity. So, if a class selector and ID selector were in conflict, the ID selector woul dbe chosen.
