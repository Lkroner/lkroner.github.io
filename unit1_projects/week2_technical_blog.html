Best Practices for Using CSS Selectors: ID vs Class

One can define your own HTML selectors in the form of Class and ID selectors. The benefit of this is that you can have the same HTML element, but present it different depending on its class. The take away statement here should be this: When deciding on which selector to use remember: ID should be used for uniquely identifying elements. Class should be used for identifying an element as being part of a class of elements.
